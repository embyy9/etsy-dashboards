PERSONAL BUDGET DASHBOARD — User Guide
========================================

Thank you for purchasing the Personal Budget Dashboard!

WHAT'S INCLUDED:
- budget-dashboard.html — The interactive dashboard (open in any browser)
- budget-tracker.xlsx — Excel version with formulas
- sample-data.json — Sample data to try the dashboard
- QUICK-START.txt — Quick setup instructions

HOW TO USE THE HTML DASHBOARD:
1. Open budget-dashboard.html in Chrome, Firefox, Safari, or Edge
2. Add transactions using the form (description, category, amount, type, date)
3. Your data saves automatically in your browser's local storage
4. Use the month filter to view specific months or "All Time" for annual overview
5. Charts update in real-time as you add data

FEATURES:
- Income & expense tracking with 15 categories
- Multiple profiles (Personal, Business, etc.) with independent data
- Household view — combined overview across all profiles
- Recurring transactions (auto-generates salary, rent, subscriptions each month)
- Category budgets with visual progress bars and overspend alerts
- Savings goals with progress tracking
- Financial health score (0-100)
- Spending calendar with daily breakdown
- Cash flow forecast (next 30 days)
- Visual charts (spending by category, monthly trends, income vs expenses)
- Annual overview with year-over-year comparison
- Summary stats (income, expenses, balance, savings rate)
- Dark mode / Light mode toggle
- Auto-detects your local currency from system locale
- CSV Export with Profile column (all profiles in Household view)
- JSON Backup & Restore — exports ALL profiles in one file
- Import from CSV/Excel spreadsheets
- Print-friendly reports
- Works 100% offline — no internet needed after first load
- All data stays on YOUR computer — completely private

PROFILES & HOUSEHOLD VIEW:
- Create multiple profiles from Settings (e.g., Personal, Business, Side Hustle)
- Each profile has its own transactions, budgets, goals, and recurring items
- Switch profiles via the dropdown in the header
- Select "Household" to see combined income/expenses across all profiles
- Household view respects the month filter — view combined data per month

RECURRING TRANSACTIONS:
- Set up recurring income/expenses from the Recurring section
- Supported frequencies: weekly, bi-weekly, monthly, quarterly, yearly
- Transactions auto-generate when you open the dashboard
- Edit a recurring item to update ALL months (past and future)
- Editing a recurring transaction from the transaction list opens the recurring editor

EXPORT & BACKUP:
- JSON Backup: Exports ALL profiles' data (transactions, budgets, goals, recurring)
- CSV Export: Current profile's transactions, or all profiles in Household view
- Restore: Accepts both multi-profile backups and single-profile exports
- Per-profile export available from Settings > Profiles > Export button
- Recommended: backup weekly or after adding many transactions

IMPORTING DATA:
- JSON Restore: Load a backup file (single-profile or multi-profile format)
- Multi-profile backups restore all profiles at once (with confirmation)
- CSV/Excel Import: Map columns to fields, preview before importing
- Supports .csv, .xlsx, and .xls files

LOADING SAMPLE DATA:
1. Open the dashboard in your browser
2. Click "Restore"
3. Select the "sample-data.json" file
4. Explore the dashboard with example data!

HOW TO USE THE EXCEL VERSION:
1. Open budget-tracker.xlsx in Excel, Numbers, or LibreOffice
2. Each month has its own tab — enter transactions there
3. The "Annual Summary" tab auto-calculates your totals
4. The "Budget Goals" tab lets you set spending limits per category

CURRENCY:
- Currency is automatically detected from your browser/system locale
- Supported: USD, EUR, GBP, JPY, INR, AUD, CAD, CHF, CNY, KRW, BRL, MXN, SGD, AED, SAR
- Change anytime from Settings — all amounts update immediately

PRIVACY:
Your financial data NEVER leaves your computer. There are no accounts,
no cloud storage, and no tracking. The dashboard works entirely offline.
